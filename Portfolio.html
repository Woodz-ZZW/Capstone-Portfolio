About My Project

Why This Topic?

I chose to create a Tennis Score Tracker because of my interest in web programming. I wanted to build a practical tool for a sport, and the specific rules of tennis scoring presented an interesting programming challenge. This project allowed me to learn and apply new web development skills.

My Goals

  • Learn to build a functional website from scratch.
  • Develop a tennis score tracker that accurately implements tennis rules.
  • Master JavaScript for interactive features and live score updates.
  • Incorporate data collection for match statistics like double faults.

The Product: Tennis Score Tracker Website

My Capstone product is a web-based application designed to keep score during a tennis match. It's built with HTML, CSS, and JavaScript.

Key features include player name input, selection of match formats (Best of 3, Best of 5, Super Tiebreak), live tracking of points, games, and sets, automatic handling of tiebreaks, double fault logging, and a comprehensive match results summary page. I chose this format to practically apply and showcase my new web development skills.

My Learning Journey

Phase 1: Getting Started (HTML & CSS - Fall 2024)

My journey began with the basics: HTML for structure and CSS for styling. It was thrilling to create my first simple webpage. Challenges included mastering advanced CSS for responsive design and the patience-testing process of debugging small code errors. YouTube tutorials and online coding platforms were my primary learning resources.

Phase 2: Diving into JavaScript (Winter 2025)

Next, I tackled JavaScript to add interactivity for real-time score updates. This was a significant step up in complexity. I started implementing data collection for statistics like double faults. My mentor introduced me to GitHub for version control, which was invaluable. I also began keeping a coding journal with screenshots and video demos to track my progress and reflect on my learning.

Phase 3: Enhancements & Complexity (Spring 2025)

I focused on improving data collection, like accurate double fault tracking, and developed the match results page. As features grew, so did the project's complexity. Managing the JavaScript state (points, games, sets, server, tiebreaks, stats) and debugging became more challenging. I increasingly relied on official programming documentation and learned to break down large tasks into smaller, manageable steps.

Challenges & Rewarding Moments

Challenges: Debugging JavaScript, understanding complex tennis rules for coding, and managing the growing complexity of the application.

Rewards: Seeing the score tracker function correctly for the first time, successfully adding planned features like the results page, and recognizing my skill improvement by comparing my early code to later versions.

Mentorship: Guidance on My Journey

I was fortunate to have a mentor who provided invaluable guidance. We had regular contact, which is a key component of the Capstone project.

My mentor helped by:

  • Reviewing my code and suggesting more efficient methods.
  • Introducing tools like GitHub for version control.
  • Discussing strategies for planning new features (e.g., statistics tracking, results page design).
  • Assisting with troubleshooting complex JavaScript bugs.
  • Providing feedback on UI/UX design ideas to improve clarity.

This mentorship taught me the importance of pre-coding planning and clear communication about technical challenges.

The Product in Action: Tennis Score Tracker

Key Features

  • Easy Setup: Input player names.
  • Match Format Selection: Best of 3, Best of 5, or Super Tiebreak.
  • Coin Toss Start: Winner chooses to serve or receive.
  • Live Score Buttons: Simple point allocation.
  • Automatic Scoreboard: Displays points, games, sets.
  • Correct Tennis Rules: Handles "Advantage" and game wins.
  • Tiebreak Logic: Automatic scoring for 6-6 games.
  • Server Indicator: Clearly shows current server.
  • Double Fault Tracking: Log double faults per player.
  • Match Results Summary: Detailed post-match overview.
  • Reset/New Match: Easy controls to restart or begin anew.
